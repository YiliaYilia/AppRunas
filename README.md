# AppRunas

工作中为控制员工账户权限，编写了这个小工具，可以使当前用户已其他用户身份运行程序，用户名、密码等信息以加密方式存储在xml文件中。

## 使用

第一次使用时，需设置使用密码，密码长度不限。

进入后，需新增任务，名称、路径、用户名及密码是必填项，各个栏位含义如下：

* 名称：任务名称，快速启动时需要使用
* 路径：程序完整路径
* 目录：程序启动时的目录
* 参数：程序启动所使用的参数
* 用户名：用户名
* 密码：用户密码
* 域：当用户为域用户时，需填写域名，否则不需填写

单击运行即可运行列表中所选任务

命令行启动：
```sh
AppRunas.exe <任务名称>
```

可制作成快捷方式，并将任务名称作为参数，即可快速启动相应任务。

## 注意

此工具运行程序时无法获取管理员权限，仅相当于以指定用户的普通权限运行程序。如确实需要提示至管理员权限，则需登录指定用户，将程序兼容性设置中`以管理员权限运行`勾选，并关闭UAC，此方法不建议使用。

配置文件`config.xml`不可在电脑间拷贝。
